# ElementOS V520 — Element Analyzer Research Instrument

This build extends the V510 Element Analyzer without changing the rest of ElementOS.

## New in V520
- Live temperature chamber (-250 °C to 3,500 °C)
- Pressure simulation (1–250 bar)
- Ionization control (-3 to +4)
- Predicted phase and environment risk state
- Condition confidence model
- Shell isolation selector
- Spectral fingerprint visualization
- Electron-shell architecture panel
- Research notebook and captured experiment states
- AI Scientist awareness of temperature, pressure, charge and phase
- Expanded live telemetry inside the atomic viewport

All outputs remain exploratory simulations and are not laboratory certification.
